Hitokotonushi is a package that can get uri from keyword.

## Installation
Use with npm.
```
$ npm install hitokotonushi
```

Use with cdn.
```
<script src="https://unpkg.com/hitokotonushi/dist/hitokotonushi.js"></script>
```
